# Sistema de Facturación – Proyecto Tienda (Python)

## Descripción
Aplicación de consola desarrollada en Python que simula un sistema de facturación para una tienda, permitiendo la gestión de clientes, facturas y distintos tipos de productos.

El proyecto fue desarrollado con fines académicos, aplicando principios de programación orientada a objetos y buenas prácticas de organización del código.

## Funcionalidades
- Crear clientes
- Generar facturas asociadas a clientes
- Agregar productos a una factura
- Calcular el total de la factura
- Buscar clientes por cédula
- Ejecución mediante menú interactivo por consola

## Estructura del proyecto
- `model/`: Entidades del sistema (Cliente, Factura, Producto)
- `crud/`: Lógica de negocio
- `ui/`: Interfaz por consola
- `tests/`: Pruebas unitarias
- `main.py`: Punto de entrada de la aplicación

## Tecnologías y conceptos aplicados
- Python
- Programación Orientada a Objetos (POO)
- Herencia y composición
- Manejo de listas y estructuras de datos
- Pruebas unitarias con `unittest`
- Organización modular del código
